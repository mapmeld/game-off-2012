Pull Command

You run an online code library called a repo.

Incoming missiles represent code changes known as pull requests.

Launch interceptors by clicking inside the game. You gain points by shooting down rejected pull requests and letting successful (merged) ones pass through.

Move the mouse over a missile to determine friend from foe.

Start in Meta or scroll down to select a repo.
